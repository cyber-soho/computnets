Key Insight: MAC Addresses

Hexadecimal is used to compress long binary strings. A MAC address (e.g., A4:5E...) is 48 bits long. Writing that in binary would take 48 digits! In Hex, it's just 12 digits.

Quick Review

Bitwise AND

The operation routers use to check destination IPs against subnet masks. If mask bit is 1, IP bit is preserved.

CIDR (/24)

Modern shorthand for subnet masks. The number represents how many bits are "1" in the mask.

NAT

Network Address Translation. Allows private IPs (RFC 1918) to access the internet via a single public IP.
