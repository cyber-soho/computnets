Flooding

Used when Destination is Unknown. Sends to all ports except source. Like asking "Is anyone here Bob?"

Forwarding

Used when Destination is Known. Sends only to the specific port. Efficient and private.

Filtering

Used when Source and Destination are on the Same Port. Drops the frame. No need to switch.

Technical Detail: The 4-Byte Insertion

In a standard Access Port, frames have no VLAN tags. The switch knows the VLAN based on the physical port configuration.

TPID (16 bits) 0x8100 Tells the device: "Hey, this is a tagged frame!"
TCI (16 bits) Pri (3) + DEI (1) + VID (12) Contains the VLAN ID (1-4094).
