Implementation Lab

Use these snippets as templates for your device configuration. Ensure interface names match your specific topology.

Step 1: Create VLANs 
enable
configure terminal
! Create VLANs
vlan 10
 name HR
vlan 20
 name Engineering
vlan 30
 name Guest
vlan 99
 name Native-Trunk
exit
Why? VLANs must exist in the switch database. If a VLAN isn't created, the switch will drop tagged frames for that ID.
Step 2: Trunks & Access 
! Access Ports (End Users)
interface fa0/1
 switchport mode access
 switchport access vlan 10
 spanning-tree portfast

! Trunk to Router (RoaS)
interface gi0/1
 switchport mode trunk
 switchport trunk native vlan 99
 switchport trunk allowed vlan 10,20,30,99
 no shutdown
Security Note: Always restrict `allowed vlan` on trunks. By default, Cisco switches allow 1-4094, which is a risk.
Router-on-a-Stick 
enable
configure terminal
interface gi0/0
 no shutdown
!
interface gi0/0.10
 encapsulation dot1q 10
 ip address 192.168.10.1 255.255.255.224
!
interface gi0/0.20
 encapsulation dot1q 20
 ip address 192.168.20.1 255.255.255.224
!
interface gi0/0.30
 encapsulation dot1q 30
 ip address 192.168.30.1 255.255.255.224
!
interface gi0/0.99
 encapsulation dot1q 99 native

Critical: The Native VLAN

Notice the command on subinterface 0.99: encapsulation dot1q 99 native.

  • The router must agree with the switch on which VLAN is native (untagged).
  • If the switch sends untagged frames (VLAN 99) and the router expects tags, traffic is dropped.
  • Always use a dedicated VLAN ID (like 99) for native traffic, never user data.

Verification Checklist

  • ✓ Layer 1: Are physical links up? (Green lights)
  • ✓ Layer 2: show vlan brief - Do VLANs exist? Are ports assigned correctly?
  • ✓ Trunks: show interfaces trunk - Is the native VLAN matching? Is encapsulation 802.1Q?
  • ✓ Layer 3: show ip route - Does the router see all connected subnets?

Common Pitfalls

Gateway Mismatch: PC has gateway .1, Router has .254. Result: Local ping works, remote fails.

Native Mismatch: Switch uses 99, Router uses 1. Result: CDP errors in console.

Missing VLAN: VLAN 20 not created on Switch 2. Result: Traffic drops at trunk.
