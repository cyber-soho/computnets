Network analogy for an IP Address
Your street address (so data packets can find your specific location).
Network analogy for DHCP
The post office that automatically assigns your address when you join a network.
Network analogy for DNS
The phone book that translates human-friendly names into routing numbers (IP addresses).
Size of the IPv4 address space
32-bit (approximately 4.3 billion unique addresses).
Notation format for IPv4 addresses
Dotted-decimal notation (e.g., 192.168.1.10).
Subnet Mask
A value that divides an IP address into network and host portions.
Default Gateway
The router address used for traffic leaving the local subnet.
Size of the IPv6 address space
128-bit (approximately 3.4 × 10³⁸ addresses).
Notation format for IPv6 addresses
Colon-hexadecimal notation.
IPv6 simplification rule for consecutive zeroes
Collapse one consecutive all-zero sequence with a double colon (::).
SLAAC (Stateless Address Autoconfiguration)
A feature allowing IPv6 devices to self-configure their IP addresses using router advertisements.
Purpose of DHCP
To automate the assignment of IP addresses and network configuration parameters to devices.
The four steps of the DHCP IPv4 process (DORA)
Discover, Offer, Request, Acknowledge.
DHCP Lease Time
The duration an IP address is rented to a device before it must be renewed.
DHCP Reservation
A permanent mapping on the DHCP server tying a specific MAC address to a specific IP address.
The three levels of the DNS Hierarchy
Root Zone, Top-Level Domain (TLD) Servers, and Authoritative Name Servers.
Recursive resolver
A DNS server that queries other servers on behalf of a client to fully resolve a domain name into an IP.
DNS A Record
Maps a domain name to an IPv4 address.
DNS AAAA Record
Maps a domain name to an IPv6 address.
DNS CNAME Record
An alias that maps one domain name to another domain name.
DNS MX Record
Specifies the mail server responsible for accepting email messages on behalf of a domain.
DNS PTR Record
Used for reverse lookups, mapping an IP address back to a domain name.
DNSSEC (DNS Security Extensions)
Adds cryptographic signatures to DNS records to prevent DNS spoofing and cache poisoning.
IETF (Internet Engineering Task Force)
The organisation that develops Internet protocol standards (such as the RFCs for IPv4, IPv6, DHCP, and DNS).
IANA (Internet Assigned Numbers Authority)
The organisation that coordinates the DNS root, IP address allocation, and protocol parameters.

1. Which network service acts like a directory, translating human-friendly names into routing numbers?

DNS provides name resolution, functioning exactly like a phonebook that maps domain names to IP addresses so browsers can connect.

2. According to IPv6 simplification rules, how would you correctly compress the address 2001:0db8:0000:0000:0000:0000:0000:0001?

Leading zeroes in a block can be dropped, and exactly one sequence of multiple all-zero blocks can be collapsed into a double colon.

3. What technology serves as a primary workaround for IPv4 address exhaustion by allowing multiple local devices to share a single public IP?

NAT translates non-routable private IPs into a single routable public IP, drastically reducing the total number of public IPs needed globally.

4. In the DHCP DORA process, what exactly happens during the 'Discover' phase?

Because the client lacks an IP and doesn't know the server's address, it must shout via broadcast to the entire local network to find a configuration source.

5. Which capability was inherently built into the design of IPv6, whereas it was only an optional add-on for IPv4?

Because security was a primary focus during the development of IPv6, cryptographic protocols for authentication and encryption are an integral part of its architecture.

6. During a recursive DNS lookup, if the local cache and the recursive resolver do not know the IP address for a domain, which server is queried first?

The recursive resolver must start at the absolute top of the hierarchy to find out which server handles the specific Top-Level Domain (like .com or .org).

7. Which DNS record type is essential if you want to perform a reverse lookup to find the domain name associated with a specific IP address?

A PTR record specifically maps an IPv4 or IPv6 address back to a canonical domain name, essentially performing the reverse function of an A or AAAA record.

8. A network administrator needs a specific printer to always receive the exact same IP address from the DHCP server. Which feature should they configure?

By mapping the printer's fixed MAC address to a specific IP, the server ensures that IP is uniquely reserved and never leased to any other device.

9. What value is used alongside an IPv4 address to determine which portion identifies the overall network and which portion identifies the specific host device?

The Subnet Mask acts as a mathematical filter, allowing devices and routers to separate the network routing prefix from the local host identifier.

10. To prevent attackers from injecting false IP records into a resolver's cache (cache poisoning), which security measure should be implemented?

DNSSEC uses cryptographic signatures to ensure that the DNS records received actually originated from the authoritative source and haven't been maliciously tampered with in transit.
